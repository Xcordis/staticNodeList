# staticNodeList

## descriptors

This project is a static linked list in C++. Its data structure is a double linked list. 
The reason for adopting this data structure is that many functions can be realized, and the time complexity is O(1).

## cmake
1. cmake -B build
2. cd build
3. cmake .
4. make


## function
1. size
2. empty
3. back
